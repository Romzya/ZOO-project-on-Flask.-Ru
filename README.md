##Zoo Management System - Flask Web Application##

Веб-приложение для управления зоопарком с разделением ролей пользователей на обычных посетителей и администраторов.

🚀 #Основные возможности#

Для всех пользователей

· Просмотр информации о зоопарке
· Просмотр списка животных с детальной информацией
· Просмотр расписания мероприятий
· Регистрация и авторизация в системе

Для администраторов

· Добавление новых животных в каталог
· Создание и управление мероприятиями
· Удаление записей
· Просмотр списка зарегистрированных пользователей

🛠 #Технологии#

· Backend: Python, Flask
· Database: SQLite
· Frontend: HTML, CSS (чистый, без фреймворков)
· Аутентификация: Werkzeug Security
· Шаблонизация: Jinja2

⚙️ #Установка и запуск#

1. Клонирование репозитория

Bash

git clone <repository-url>
cd zoo_project
1. Создание виртуального окружения

Bash

python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
1. Установка зависимостей

Bash

pip install -r requirements.txt
1. Запуск приложения

Bash

python app.py
1. Открыть в браузере

http://127.0.0.1:5522

🔐 #Функции безопасности#

· Хэширование паролей с использованием Werkzeug
· Защита сессий
· Проверка прав доступа для административных функций
· SQL-инъекции предотвращены через параметризованные запросы

📋 #Требования#

· Python 3.7+
· Flask 2.0+
· SQLite3 (встроен в Python)

🌐 #Особенности интерфейса#

· Адаптивный дизайн (работает на мобильных устройствах)
· Интуитивно понятная навигация
· Визуальная обратная связь через flash-сообщения
· Подтверждение действий (удаление записей)

🗄 #База данных#

При первом запуске автоматически создается база данных zoo.db со следующими таблицами:

· users - информация о пользователях (обычные и администраторы)
· animals - информация о животных зоопарка
· events - расписание мероприятий
